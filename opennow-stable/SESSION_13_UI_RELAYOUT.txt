SESSION 13 - UI ReLayout (Instagram-style bottom tab bar)
==========================================================
Date: 2026-02-27

GOAL
----
Redesign the app UI to match an Instagram-style bottom tab bar with:
- 3 tabs at bottom: Catalog (left), Library (middle), Settings (right)
- 2 games per row in both Catalog and Library
- User profile/logout moved from top navbar into Settings page

CHANGES MADE
------------

1. src/renderer/src/components/Navbar.tsx
   - Removed: user, subscription, onLogout props
   - Removed: All modal code (time/storage details)
   - Removed: Horizontal top bar layout
   - Added: navbar-tabs div with 3 .navbar-tab buttons (icon + label)
   - Active session resume is now a full-width strip at top of navbar
   - Now a bottom bar component only

2. src/renderer/src/components/SettingsPage.tsx
   - Added imports: User, LogOut icons; AuthUser, SubscriptionInfo types
   - Added props: user, subscription, onLogout
   - Added getTierDisplay() helper function
   - Added user profile card at top of settings page render:
     * Avatar image or fallback icon
     * Display name, email, tier badge, remaining hours
     * Sign out button (logout)

3. src/renderer/src/App.tsx
   - Removed user, subscription, onLogout props from Navbar call
   - Added user, subscription, onLogout props to SettingsPage call

4. src/renderer/src/styles.css
   - --navbar-h changed: 48px -> 64px (bottom tab bar taller)
   - .navbar: position fixed bottom (was top), border-top (was border-bottom)
     flex-direction: column, removed padding
   - Added .navbar-tabs: flex row, space-around
   - Added .navbar-tab: flex column, icon + label, accent when active
   - .navbar-session-resume: full-width strip, no border-radius, sits above tabs
   - .main-content: margin-bottom: var(--navbar-h) instead of margin-top
   - .auth-refresh-notice: top: 10px instead of calc(navbar-h + 10px)
   - .game-grid: always repeat(2, 1fr) - removed all responsive breakpoints
   - Added .settings-profile and sub-classes for the user profile card in settings
   - controller-mode --navbar-h: 68px
